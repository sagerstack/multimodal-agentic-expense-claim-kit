## Decisions Made

1. **QwQ-32B as primary model** - Reasoning model provides Type B reasoning tokens (visible thinking process) needed for the thinking panel UI. Gemini Flash retained as fallback for 402 quota errors.

2. **Temperature 0.3 instead of 0.7** - Reasoning models perform better with lower temperature (less randomness, more consistent structured reasoning). Made configurable via OPENROUTER_LLM_TEMPERATURE env var.

3. **Max tokens 8192 (doubled from 4096)** - QwQ-32B produces verbose reasoning output with explicit thinking steps. Higher token limit prevents truncation of reasoning chains.

4. **getClaimSchema as first tool** - Positioning schema discovery first signals to the agent that schema awareness is foundational. Enables future schema-driven system prompt (Plan 02.5-03).

5. **information_schema query for schema discovery** - Standard SQL pattern for runtime schema introspection. Returns column_name, data_type, is_nullable, column_default for all columns in claims and receipts tables.
