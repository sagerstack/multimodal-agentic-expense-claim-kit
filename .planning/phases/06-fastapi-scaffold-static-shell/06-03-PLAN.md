  <action>
    Create `tests/test_web_pages.py` with smoke tests using FastAPI's `TestClient`.

    IMPORTANT: The TestClient needs the FastAPI app, but the app's lifespan calls `getCompiledGraph()` which requires a real Postgres connection. The tests must mock the lifespan to avoid needing a database.

    1. Create a test fixture that patches the lifespan to skip graph/checkpointer initialization:
       ```python
       @pytest.fixture
       def client():
           """TestClient with mocked lifespan (no DB required)."""
           from unittest.mock import AsyncMock, patch
           from contextlib import asynccontextmanager

           @asynccontextmanager
           async def mockLifespan(app):
               app.state.graph = AsyncMock()
               app.state.checkpointerCtx = AsyncMock()
               yield

           with patch("agentic_claims.web.main.lifespan", mockLifespan):
               from agentic_claims.web.main import app
               from starlette.testclient import TestClient
               with TestClient(app) as c:
                   yield c
       ```

    2. Write these test functions (CamelCase names per project convention):
