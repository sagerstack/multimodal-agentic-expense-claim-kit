       - Top section (below top nav, mt-16):
         - "Luminous Intelligence" header with bolt icon in gradient background.
         - "Expense Engine Active" subtitle.
         - "New Expense" button with gradient and hover glow.
       - Navigation items (`<nav>` with `space-y-1`):
         - **Chat AI** — icon: `chat_spark`, href: `/`, activePage: `chat`
         - **Dashboard** — icon: `insert_chart`, href: `/dashboard`, activePage: `dashboard`
         - **Audit Log** — icon: `verified_user`, href: `/audit`, activePage: `audit`
         - **Review** — icon: `receipt_long`, href: `#` (dynamic URL), activePage: `review`
       - Each nav item uses Jinja2 conditional for active state:

         - Inactive: `text-[#dee5ff] opacity-50 hover:bg-[#192540] hover:opacity-100`

       - Bottom section (mt-auto):
         - "System Status" link with `sensors` icon.
         - Intelligence pulse dot (`.intelligence-pulse` CSS class) next to "System Status" text.

       - `<main class="md:ml-64 pt-16 h-screen overflow-hidden bg-surface">`.
       - `{% block content %}{% endblock %}` inside main.

    6. **Block definitions:**
       - `{% block title %}{% endblock %}` — page title suffix.
       - `{% block content %}{% endblock %}` — page content.
       - `{% block scripts %}{% endblock %}` — page-specific scripts (at end of body).

    IMPORTANT: Copy the exact Tailwind classes from Stitch HTML. Do NOT simplify or reinterpret the design — match it precisely. Use hex color values in brackets (e.g., `bg-[#060e20]`) where the Stitch HTML uses them, and semantic token names (e.g., `text-on-surface`) where Stitch uses them.
