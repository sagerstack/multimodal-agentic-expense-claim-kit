  **Dockerfile (BUG-006):**
  Add `ca-certificates` system package and upgrade `certifi` Python package. The current Dockerfile installs only `curl` in the apt-get step.

  Change the apt-get line from:
  ```
  RUN apt-get update && apt-get install -y curl && rm -rf /var/lib/apt/lists/*
  ```
  to:
  ```
  RUN apt-get update && apt-get install -y curl ca-certificates && rm -rf /var/lib/apt/lists/*
  ```

  ```
  RUN pip install --no-cache-dir --upgrade certifi

  ```
  ENV REQUESTS_CA_BUNDLE=/etc/ssl/certs/ca-certificates.crt
