    **2. Create 4 MCP servers using FastMCP:**

    All MCP servers follow the same pattern:
    - Python file with FastMCP decorators
    - Dockerfile based on python:3.11-slim
    - requirements.txt with dependencies
    - SSE transport on 0.0.0.0:8000 (Docker networking)
    - Health check endpoint

    **RAG MCP Server (`mcp_servers/rag/`):**
    - `server.py`: FastMCP server with tools:
      - `searchPolicies(query: str, limit: int = 5) -> list[dict]`: Semantic search against Qdrant "expense_policies" collection. Returns list of {text, file, score, category}.
      - `getPolicyByCategory(category: str) -> list[dict]`: Filter policies by category metadata.
    - Requires: `fastmcp`, `qdrant-client`, `sentence-transformers`
    - Environment: QDRANT_URL (from env), COLLECTION_NAME (from env), EMBEDDING_MODEL (from env)
    - Initialize SentenceTransformer at startup (download model on first run)
    - Dockerfile: Install deps, copy server.py, CMD runs server with SSE transport

    **DB MCP Server (`mcp_servers/db/`):**
    - `server.py`: FastMCP server with tools:
      - `executeQuery(query: str) -> list[dict]`: Execute read-only SQL query against Postgres. Returns rows as dicts.
      - `insertClaim(claimNumber: str, employeeId: str, status: str, totalAmount: float, currency: str = "SGD") -> dict`: Insert a claim, return the new record.
      - `updateClaimStatus(claimId: int, newStatus: str, actor: str) -> dict`: Update claim status AND insert audit_log entry. Return updated claim.
      - `getClaimWithReceipts(claimId: int) -> dict`: Get claim with all linked receipts.
    - Requires: `fastmcp`, `psycopg[binary]`
    - Environment: DATABASE_URL (postgresql connection string from env)
    - Use psycopg3 async connection for queries
    - Dockerfile: Install deps, copy server.py, CMD runs server with SSE transport

    **Currency MCP Server (`mcp_servers/currency/`):**
    - `server.py`: FastMCP server with tools:
      - `convertCurrency(amount: float, fromCurrency: str, toCurrency: str = "SGD") -> dict`: Convert using Frankfurter API (https://api.frankfurter.dev/v1/latest?from={from}&to={to}). Returns {originalAmount, originalCurrency, convertedAmount, convertedCurrency, rate, date}.
      - `getSupportedCurrencies() -> list[str]`: List supported currencies from Frankfurter API.
    - Requires: `fastmcp`, `httpx`
    - No env vars needed (Frankfurter is free, no API key)
    - Dockerfile: Install deps, copy server.py, CMD runs server with SSE transport

    **Email MCP Server (`mcp_servers/email/`):**
    - `server.py`: FastMCP server with tools:
      - `sendEmail(to: str, subject: str, body: str) -> dict`: Send email via SMTP. Returns {success, messageId}. For local dev, log the email instead of actually sending if SMTP_HOST is "mailhog" or "localhost".
      - `sendClaimNotification(to: str, claimNumber: str, status: str, message: str) -> dict`: Convenience wrapper for claim-related notifications. Builds a formatted email body.
    - Requires: `fastmcp`, `aiosmtplib` (for async SMTP)
    - Environment: SMTP_HOST, SMTP_PORT, SMTP_USER, SMTP_PASSWORD (from env)
    - For local dev, use MailHog (will add to docker-compose) or just log emails to stdout
    - Dockerfile: Install deps, copy server.py, CMD runs server with SSE transport
