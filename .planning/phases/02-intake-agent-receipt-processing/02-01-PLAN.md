    **2. Create SQLAlchemy ORM models in `src/agentic_claims/infrastructure/database/models.py`:**
    - Use SQLAlchemy 2.0 Mapped style (DeclarativeBase)
    - CamelCase class names: `Claim`, `Receipt`, `AuditLog`
    - snake_case column names (database convention): `claim_number`, `employee_id`, `total_amount`, etc.
    - `Base = DeclarativeBase()` at module level for Alembic target_metadata

    **Claim model:**
    - `id`: Integer, primary key, autoincrement
    - `claimNumber`: Mapped[str] = mapped_column(String(50), unique=True, nullable=False, name="claim_number")
    - `employeeId`: Mapped[str] = mapped_column(String(50), nullable=False, index=True, name="employee_id")
    - `status`: Mapped[str] = mapped_column(String(20), nullable=False) -- values: draft, submitted, approved, returned, escalated
    - `totalAmount`: Mapped[Decimal] = mapped_column(Numeric(10, 2), nullable=False, name="total_amount")
    - `currency`: Mapped[str] = mapped_column(String(3), nullable=False, server_default="SGD")
    - `submissionDate`: Mapped[Optional[datetime]] = mapped_column(DateTime(timezone=True), nullable=True, name="submission_date")
    - `approvalDate`: Mapped[Optional[datetime]] = mapped_column(DateTime(timezone=True), nullable=True, name="approval_date")
    - `createdAt`: Mapped[datetime] = mapped_column(DateTime(timezone=True), server_default=text("now()"), name="created_at")
    - `updatedAt`: Mapped[datetime] = mapped_column(DateTime(timezone=True), server_default=text("now()"), onupdate=text("now()"), name="updated_at")
    - Relationship: `receipts` -> list[Receipt] (back_populates)
    - Relationship: `auditLogs` -> list[AuditLog] (back_populates)
    - `__tablename__ = "claims"`

    **Receipt model:**
    - `id`: Integer, primary key
    - `claimId`: Mapped[int] = mapped_column(ForeignKey("claims.id", ondelete="CASCADE"), nullable=False, index=True, name="claim_id")
    - `receiptNumber`: Mapped[str] = mapped_column(String(50), nullable=False, name="receipt_number")
    - `merchant`: Mapped[str] = mapped_column(String(200), nullable=False)
    - `date`: Mapped[date] = mapped_column(Date, nullable=False)
    - `totalAmount`: Mapped[Decimal] = mapped_column(Numeric(10, 2), nullable=False, name="total_amount")
    - `currency`: Mapped[str] = mapped_column(String(3), nullable=False)
    - `imagePath`: Mapped[Optional[str]] = mapped_column(String(500), nullable=True, name="image_path") -- filesystem path for receipt image (DATA-04)
    - `lineItems`: Mapped[dict] = mapped_column(JSONB, nullable=False, name="line_items") -- [{description, amount, glCode, category}, ...]
    - `createdAt`, `updatedAt` same as Claim
    - Relationship: `claim` -> Claim (back_populates)
    - `__tablename__ = "receipts"`

    **AuditLog model:**
    - `id`: Integer, primary key
    - `claimId`: Mapped[int] = mapped_column(ForeignKey("claims.id", ondelete="CASCADE"), nullable=False, index=True, name="claim_id")
    - `action`: Mapped[str] = mapped_column(String(50), nullable=False) -- status_change, field_update, comment_added
    - `oldValue`: Mapped[Optional[str]] = mapped_column(Text, nullable=True, name="old_value")
    - `newValue`: Mapped[Optional[str]] = mapped_column(Text, nullable=True, name="new_value")
    - `actor`: Mapped[str] = mapped_column(String(100), nullable=False) -- employee_id or "system"
    - `timestamp`: Mapped[datetime] = mapped_column(DateTime(timezone=True), server_default=text("now()"))
    - Relationship: `claim` -> Claim (back_populates)
    - `__tablename__ = "audit_log"`

    NOTE on naming: Use CamelCase for Python attribute names with explicit `name=` parameter for snake_case column names in the database. This matches the project's CamelCase convention while keeping standard SQL naming in the DB.

    **1. Create `src/agentic_claims/infrastructure/openrouter/client.py`:**

    Class: `OpenRouterClient`
    - Constructor takes `Settings` instance (dependency injection, not creating its own)
    - Creates `AsyncOpenAI` client with `api_key=settings.openrouter_api_key` and `base_url=settings.openrouter_base_url`
    - Stores `settings.openrouter_max_retries` and `settings.openrouter_retry_delay`

    Method: `async def callLlm(self, messages: list[dict], model: str | None = None) -> str`
    - Uses model param or falls back to `settings.openrouter_model_llm`
    - Calls `self.client.chat.completions.create(model=model, messages=messages)`
    - Returns `response.choices[0].message.content`
    - Retry logic: loop `max_retries` times, on any exception sleep `retry_delay` seconds then retry. On final failure, re-raise the exception.

    Method: `async def callVlm(self, text: str, imageUrl: str, model: str | None = None) -> str`
    - Builds multimodal message with text and image_url content parts
    - Uses model param or falls back to `settings.openrouter_model_vlm`
    - Delegates to `callLlm` with constructed messages and model

    IMPORTANT: Do NOT use default parameter values for model names, retry count, or delay. All values must come from Settings (loaded from .env).

    **2. Create `__init__.py`** for openrouter package (empty).

    **3. Write `tests/test_openrouter.py`:**
    - Test that `OpenRouterClient` can be instantiated with test settings
    - Test `callLlm` constructs correct API call (mock AsyncOpenAI -- use `unittest.mock.AsyncMock` to patch `self.client.chat.completions.create`)
    - Test retry logic: mock create to fail twice then succeed on third call. Verify it was called 3 times. Use `unittest.mock.patch("asyncio.sleep")` to skip actual delays.
    - Test retry exhaustion: mock create to always fail. Verify exception is raised after max_retries attempts.
    - Test `callVlm` builds correct multimodal message format (verify the messages list structure passed to callLlm)

    All tests are unit tests using mocks -- no real OpenRouter API calls.
