    1. Create `src/agentic_claims/web/routers/dashboard.py`:
       - Create an async SQLAlchemy session factory using Settings.postgres_dsn_async (use `create_async_engine` + `async_sessionmaker` from sqlalchemy.ext.asyncio). Define this at module level or as a helper.

       - `GET /dashboard` -> **HTML page handler** (moved from pages.py). This serves the dashboard HTML page with full server-side data:
         - Import `getCurrentUser` from auth.py, check `role == "reviewer"` (redirect to `/` if not)
         - Query KPIs (pending/autoApproved/escalated counts) and claims list server-side
         - Pass to template context: `kpis`, `claims`, `userRole`, `displayName`, `employeeId`, `username`, `activePage="dashboard"`, `threadId`, `claimId`
         - Render `templates/dashboard.html`

       - `GET /api/dashboard/kpis` -> returns JSON: `{pending: int, autoApproved: int, escalated: int}`
         Query: COUNT claims grouped by status. Map statuses: "submitted"/"pending" -> pending, "approved" -> autoApproved, "escalated"/"flagged" -> escalated.
       - `GET /api/dashboard/claims` -> returns JSON list of claims with fields: `{id, claimNumber, employeeId, employeeName, category, totalAmount, currency, status, createdAt}`
         Join claims with users table on employee_id to get display_name as employeeName. Order by created_at DESC. If no user match, show employee_id as name.
         Extract category from the receipt's line_items JSONB or from intake_findings if available.
       - `GET /api/dashboard/efficiency` -> returns JSON: `{hourlyRates: [{hour: str, rate: float}]}` 
         Calculate auto-approval rate by hour from claims created_at and status. Group by hour of day, calculate (approved count / total count) per hour. Return last 24 hours of data.

    3. Include the dashboard router in main.py (`app.include_router(dashboardRouter)`).

       - `testKpisEndpointReturnsCorrectShape`: Mock DB session, verify JSON shape
       - `testClaimsEndpointReturnsList`: Mock DB session, verify list response
       - `testEfficiencyEndpointReturnsHourlyRates`: Mock DB session, verify hourly rate shape
       - Use TestClient with mocked auth session
