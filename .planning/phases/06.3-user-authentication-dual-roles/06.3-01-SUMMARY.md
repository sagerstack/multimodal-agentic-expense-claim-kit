### Task 2: Auth module, login/logout endpoints, login template, tests
- Created `src/agentic_claims/web/auth.py`
  - `authenticateUser(username, password)` — async DB query + bcrypt verification
  - `requireAuth(request)` — returns 302 redirect or None
  - `getCurrentUser(request)` — extracts user dict from session, raises 401 if missing
  - `requireRole(role)` — dependency factory for role enforcement
- Created `src/agentic_claims/web/routers/auth.py`
  - `GET /login` — renders login.html (redirects to role default if already authenticated)
  - `POST /login` — validates credentials, sets session, redirects by role (user→/, reviewer→/dashboard)
  - `GET /logout` — clears session, redirects to /login
- Updated `src/agentic_claims/web/main.py`
  - Added `AuthMiddleware` (BaseHTTPMiddleware) — exempts /login, /logout, /static/*
  - Middleware registration order: `AuthMiddleware` first, `SessionMiddleware` second (LIFO ensures session is populated before auth check)
  - Included authRouter
- Created `templates/login.html`
  - Standalone page (no base.html extension)
  - Matches Stitch dark theme: glass-morphism panel, neon glow, gradient "Expense AI" branding
  - Fields: username, password, remember device checkbox, Sign In button
  - Error message shown inline when login fails, username preserved
