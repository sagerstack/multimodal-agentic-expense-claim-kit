`src/agentic_claims/agents/intake/tools/searchPolicies.py`:
```python
@tool
async def searchPolicies(query: str, limit: int = 5) -> list[dict]:
    """Search expense policies using semantic search. Returns relevant policy clauses with section references and relevance scores.

    Args:
        query: Natural language query about expense policies
        limit: Maximum number of results (default 5)
    """
```
- Call `mcpCallTool(settings.rag_mcp_url, "searchPolicies", {"query": query, "limit": limit})`
- Return results directly (list of dicts with text, file, category, section, score)
- On error, return `[{"error": str(e)}]`

`src/agentic_claims/agents/intake/tools/convertCurrency.py`:
```python
@tool
async def convertCurrency(amount: float, fromCurrency: str, toCurrency: str = "SGD") -> dict:
    """Convert a currency amount to SGD using live exchange rates from European Central Bank.

    Args:
        amount: Amount to convert
        fromCurrency: Source currency code (e.g., USD, EUR, GBP)
        toCurrency: Target currency code (default SGD)
    """
```
- Call `mcpCallTool(settings.currency_mcp_url, "convertCurrency", {"amount": amount, "fromCurrency": fromCurrency, "toCurrency": toCurrency})`
- MCP returns list of content items. Parse the first content item's text (may be JSON string).
- Return conversion dict with originalAmount, convertedAmount, rate, date
- On error, return `{"error": str(e)}`

`src/agentic_claims/agents/intake/tools/submitClaim.py`:
```python
@tool
async def submitClaim(claimNumber: str, employeeId: str, totalAmount: float, currency: str = "SGD", originalCurrency: str = "", originalAmount: float = 0.0) -> dict:
    """Submit a validated expense claim to the database.

    Args:
        claimNumber: Unique claim identifier
        employeeId: Employee ID of the claimant
        totalAmount: Total amount in SGD (after conversion if foreign currency)
        currency: Currency code (default SGD)
        originalCurrency: Original currency if foreign (empty if SGD)
        originalAmount: Original amount before conversion (0 if SGD)
    """
```
- Step 1: Call `mcpCallTool(settings.db_mcp_url, "insertClaim", {"claimNumber": claimNumber, "employeeId": employeeId, "status": "submitted", "totalAmount": totalAmount, "currency": currency, "originalCurrency": originalCurrency, "originalAmount": originalAmount})` to create the claim record.
- Step 2: Parse the claim result to extract the new claim's `id` (foreign key).
- Step 3: Call `mcpCallTool(settings.db_mcp_url, "insertReceipt", {"claimId": claimId, "merchantName": merchantName, "receiptDate": receiptDate, "totalAmount": totalAmount, "currency": currency, "originalCurrency": originalCurrency, "originalAmount": originalAmount})` to create the receipt record linked to the claim.
- Return dict with both claim and receipt records: `{"claim": claimRecord, "receipt": receiptRecord}`
- On error in either step, return `{"error": str(e)}`

**NOTE**: The submitClaim tool signature should accept additional receipt fields (merchantName, receiptDate, etc.) or accept a single `receiptData: dict` parameter containing the extracted receipt fields.

def upgrade():
    # Add to claims table

    op.add_column("claims", sa.Column("original_amount", sa.Numeric(10, 2), nullable=True))
    op.add_column("claims", sa.Column("converted_amount_sgd", sa.Numeric(10, 2), nullable=True))

    # Add to receipts table
    op.add_column("receipts", sa.Column("original_currency", sa.String(3), nullable=True))
    op.add_column("receipts", sa.Column("original_amount", sa.Numeric(10, 2), nullable=True))
    op.add_column("receipts", sa.Column("converted_amount_sgd", sa.Numeric(10, 2), nullable=True))

def downgrade():
    # Remove from receipts
    op.drop_column("receipts", "converted_amount_sgd")
    op.drop_column("receipts", "original_amount")
    op.drop_column("receipts", "original_currency")

    # Remove from claims
    op.drop_column("claims", "converted_amount_sgd")
    op.drop_column("claims", "original_amount")
    op.drop_column("claims", "original_currency")
