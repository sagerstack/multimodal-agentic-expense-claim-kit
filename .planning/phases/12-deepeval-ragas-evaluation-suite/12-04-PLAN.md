<task type="auto">
  <name>Task 2: Build run_eval.py orchestrator with CLI args</name>
  <files>eval/run_eval.py</files>
  <action>
Create `eval/run_eval.py` as the main entry point.

**CLI arguments** (use argparse -- stdlib, no extra deps):
- `--skip-capture`: Skip browser capture phase, load results from `eval/results/` JSON files. For re-scoring without re-running Playwright.
- `--benchmark ER-XXX`: Run a single benchmark instead of all 20. Works with or without `--skip-capture`.
- `--skip-push`: Skip Confident AI dashboard push (useful for local debugging).
- `--verbose`: Print detailed metric reasons for each benchmark.

**Pipeline steps** (run sequentially):

```
1. CAPTURE (skip if --skip-capture)
   For each benchmark:
     a. Build subagent prompt via buildCapturePrompt()
     b. Print the prompt to stdout with clear delimiters:
        "=== CAPTURE PROMPT FOR {benchmarkId} ==="
        {prompt}
        "=== END PROMPT ==="
        "Please paste the captured JSON output below (or type 'skip' to skip this benchmark):"
     c. Read the captured JSON from stdin
     d. Save to eval/results/{benchmarkId}.json via saveCapturedResult()
   NOTE: The actual Playwright automation is driven by a Claude subagent externally.
   run_eval.py provides the prompts and collects the results.
   In future, this can be automated via Claude API programmatic subagent invocation.

2. LOAD
   Load all captured results from eval/results/ via loadAllCapturedResults()
   Filter to single benchmark if --benchmark specified

3. ENRICH
   For each captured result:
     enrichCapturedResult(result, config.dbUrl)
   Save enriched results back to eval/results/{benchmarkId}.json

4. SCORE
   For each captured result:
     a. Get benchmark definition from dataset
     b. Get metrics via getMetricsForBenchmark()
     c. Build LLMTestCase via buildTestCase()
     d. Collect into lists for evaluate()

   Call deepeval's evaluate():
     from deepeval import evaluate
     evaluationResult = evaluate(
         test_cases=testCases,
         metrics=allMetrics,
         identifier=f"MMGA-Run-{datetime.now().isoformat()}",
         hyperparameters={
             "judgeModel": "gpt-4o",
             "appVersion": "phase-12",
         },
     )

   NOTE: evaluate() in deepeval 3.x returns EvaluationResult object.
   Access results via evaluationResult.test_results or iterating evaluationResult.

5. REPORT
   Build results list from evaluation output
   Call scoring functions:
     categoryScores = calculateCategoryScores(results)
     overallScore = calculateWeightedOverall(categoryScores)
     primaryTargets = checkPrimaryTargets(results)
     printSummaryTable(results, categoryScores, overallScore)

   Confident AI push happens automatically via evaluate() if CONFIDENT_AI_API_KEY is set.
   If --skip-push, set env var DEEPEVAL_TELEMETRY_OPT_OUT=1 before evaluate().
```

**Error handling:**
- If capture JSON is invalid, log error and skip benchmark
- If enrichment fails, continue with partial data (warn in summary)
- If a metric fails to evaluate, record score=0.0 with reason "metric error: {exception}"
- If no captured results found and --skip-capture, exit with error message

**Main function:**
```python
async def main():
    args = parseArgs()
    config = getEvalConfig()
    # ... pipeline steps
```

Entry point:
```python
if __name__ == "__main__":
    import asyncio
    asyncio.run(main())
```

CamelCase naming throughout. Async for enrichment (DB queries are async).
