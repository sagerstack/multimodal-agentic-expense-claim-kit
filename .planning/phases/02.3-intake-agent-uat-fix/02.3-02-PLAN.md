  <name>Task 2: Implement 402 fallback retry in VLM extraction and agent LLM</name>

  <action>
  **Part A: VLM 402 fallback in extractReceiptFields.py**

  Wrap the VLM call (Step 5: `response = await vlm.ainvoke([message])`) in a try/except that catches the 402 error and retries with the fallback model.

  The OpenAI SDK raises `openai.APIStatusError` (or similar) for HTTP errors. The 402 error from OpenRouter contains "This request requires more credits" in the message. Catch broadly and check for "402" or "credits" in the error string.

  Implementation pattern:
  ```python
  import logging
  logger = logging.getLogger(__name__)

  # Step 5: Call VLM (with 402 fallback)
  try:
      response = await vlm.ainvoke([message])
  except Exception as e:
      errorStr = str(e)
      if "402" in errorStr or "credits" in errorStr.lower() or "quota" in errorStr.lower():
          logger.warning(
              "Primary VLM model returned 402, falling back",
              extra={
                  "primaryModel": settings.openrouter_model_vlm,
                  "fallbackModel": settings.openrouter_fallback_model_vlm,
                  "error": errorStr[:200],
              }
          )
          fallbackVlm = ChatOpenAI(
              model=settings.openrouter_fallback_model_vlm,
              base_url=settings.openrouter_base_url,
              api_key=settings.openrouter_api_key,
              temperature=0.0,
          )
          response = await fallbackVlm.ainvoke([message])
      else:
          raise  # Re-raise non-402 errors
  ```

  Add `import logging` and create `logger = logging.getLogger(__name__)` at module level if not already present.

  **Part B: Chat LLM 402 fallback in node.py**

  In getIntakeAgent(), wrap the agent invocation in intakeNode() with a similar 402 fallback pattern. If the primary LLM fails with 402 during the ReAct loop, recreate the agent with the fallback model and retry.

  In intakeNode():
  ```python
  try:
      result = await agent.ainvoke(agentInput)
  except Exception as e:
      errorStr = str(e)
      if "402" in errorStr or "credits" in errorStr.lower() or "quota" in errorStr.lower():
          logger.warning(
              "Primary LLM returned 402, falling back",
              extra={
                  "primaryModel": settings.openrouter_model_llm,
                  "fallbackModel": settings.openrouter_fallback_model_llm,
                  "error": errorStr[:200],
              }
          )
          fallbackAgent = getIntakeAgent(useFallback=True)
          result = await fallbackAgent.ainvoke(agentInput)
      else:
          raise
  ```

  Modify getIntakeAgent to accept an optional `useFallback: bool = False` parameter. When True, use `settings.openrouter_fallback_model_llm` instead of `settings.openrouter_model_llm`.

  Add `import logging` and `logger = logging.getLogger(__name__)` at module level in node.py if not already present.
