## Decisions Made

**1. Filter third-party loggers globally in setupLogging()**
- Apply filtering in setupLogging() after all handlers are attached
- Filter openai, httpx, httpcore, urllib3, asyncio to WARNING level
- Prevents debug payloads from OpenAI SDK polluting Seq with unstructured logs

**2. Detect 402 errors via multiple signals**
- Check for "402" in error string (HTTP status code)
- Check for "credits" or "quota" in error message (OpenRouter payment language)
- Broad detection pattern ensures we catch quota errors even if error format changes

**3. Fallback model selection via useFallback parameter**
- getIntakeAgent() accepts useFallback: bool = False
- When True, uses settings.openrouter_fallback_model_llm instead of primary
- Clean separation of fallback logic without code duplication

**4. Log fallback events with structured extra fields**
- Log at WARNING level when fallback occurs
- Include primary_model, fallback_model, and error in extra fields
- Enables Seq queries for fallback frequency and error patterns
