## Decisions Made

**Field Mapping Architecture:**
- CLAIM_FIELD_MAP: Maps agent claim vocabulary to MCP parameters (claimantId -> employeeId, amountSgd -> totalAmount)
- RECEIPT_FIELD_MAP: Maps agent receipt vocabulary to MCP parameters (date -> receiptDate, totalAmount -> receiptTotalAmount)
- merchant maps to merchant (NOT receiptMerchant) - MCP parameter names are canonical

**Auto-Generation Strategy:**
- claimNumber: CLAIM-{001-999} if not provided by agent
- receiptNumber: REC-{001-999} if receipt data exists but no receiptNumber
- status: "pending" if not provided by agent

**Unmapped Key Handling:**
- Log warnings for any agent keys not in mapping dictionaries
- Enables debugging and evolution as agent vocabulary changes
