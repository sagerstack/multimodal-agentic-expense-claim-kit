**GREEN phase:** Create `src/agentic_claims/web/employeeIdExtractor.py` with `extractEmployeeId(text: str) -> str | None`.

Strategy:
1. Search for explicit patterns first: `EMP-\d+`, `EMP\d+`, or any `[A-Z]{2,5}-\d+` pattern
2. Search for contextual patterns: "employee id" or "ID:" followed by alphanumeric string
3. Search for "employee" followed by a number
4. Return None if no pattern matches
5. If multiple matches, return the LAST one (user correction pattern)
6. Normalize: uppercase any alphabetic prefix

Run tests — they should all PASS.
