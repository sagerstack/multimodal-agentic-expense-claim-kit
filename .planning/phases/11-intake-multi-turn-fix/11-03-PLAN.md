For each file, add `from agentic_claims.core.logging import logEvent` and convert every `logger.info()`, `logger.warning()`, `logger.error()`, and `logger.debug()` call to `logEvent()`.

Pattern for conversion:
```python
# Before:
logger.info("complianceNode started", extra={"claimId": claimId})

# After:
logEvent(
    logger,
    "compliance.started",
    logCategory="agent",
    actorType="agent",
    agent="compliance",
    claimId=claimId,
    message="Compliance agent started",
)
```

For `logger.warning()` and `logger.error()`, pass `level=logging.WARNING` or `level=logging.ERROR` to logEvent.
